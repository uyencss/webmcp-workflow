Dưới đây là một hàm Python ngắn gọn sử dụng phương pháp đệ quy để tính giai thừa của một số nguyên:

Python

```
def tinh_giai_thua(n):
    if n == 0 or n == 1:
        return 1
    return n * tinh_giai_thua(n - 1)

# Ví dụ cách sử dụng:
# print(tinh_giai_thua(5))  # Kết quả sẽ là 120
```

Nếu bạn thích cách tiếp cận vòng lặp (iterative) để tránh giới hạn đệ quy với các số lớn, bạn có thể viết như sau:

Python

```
def tinh_giai_thua_vong_lap(n):
    ket_qua = 1
    for i in range(2, n + 1):
        ket_qua *= i
    return ket_qua
```

*(Mẹo nhỏ: Trong thực tế, nếu không bắt buộc phải tự viết hàm, bạn có thể dùng luôn `math.factorial(n)` từ thư viện chuẩn `math` của Python cho nhanh và tối ưu nhất nhé).*
